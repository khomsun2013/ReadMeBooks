There are millions of books in the world, written in hundreds of different languages, filled with stories of adventures, mysteries, science, facts, and so much more. The problem with so many books is that it can be quite difficult to choose the right one because while there are millions of stories to read, not all might fit a reader’s preference, since every reader has their own personalized style and taste. Therefore, how do you tell if you will like a book before you even read it? Well, you can’t judge a book by its cover nor will you want to waste time reading an entire book just to realize that you didn’t like it. This can become quite an issue when it comes to choosing a book to read, may it be a new book in a new genre or a book similar to one you love. Therefore, we propose a book recommendation system based on data set (BX-CSV-Dump).


Objective:

1. Create the book recommendation system based on Users and Books Rating.

2. Develop the web application using Machine Learning Techniques such as KNN-NearestNeighbors based on Users and Books Rating, and Word2Vec Model.

3. Provide the recommend books for the users based on the users rating dataset.

4. Present the similar books that are related to the book.

5. Exhibit the pilot project that apply the Machine Learning Technique for any recommendation systems.

6. This project will the business to make more revenue, because the results from the recommendation are high accuracy.


Who are get the benifits from the project:

1. The general users who are members of the Book recommendation system.

2. The new users, typical person who are interesting and want to get the recommendations from the project.

3. The wormbooks who are interesting for discovering the similar books and recommend books from the project.

4. The business organization that want to create or improve their recommendation system.

Machine Learning Techiques:

1. KNN-NearestNeighbors based on User Ratings. The system will find the recommendation books by search the users who have the similar ratings. Since the users who are similar should like the similar books. Hence the system will gather the books from similar users provide to the user by using Cosine or Correlation values.

2. Word2Vec Model. The project apply the Word2Vec technique to search the similar books by generate vector for each bookId. And also calculate the similar books by compare the vectors.
